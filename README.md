#  Zepto Sales Dashboard using Microsoft Excel

## 📌 Project Overview

The **Zepto Sales Dashboard** is an interactive Business Intelligence dashboard developed using **Microsoft Excel**. It transforms raw sales data into meaningful insights using Pivot Tables, Pivot Charts, KPI Cards, and Slicers.

The dashboard enables users to analyze sales performance, customer behavior, product categories, city-wise sales, and delivery performance through interactive visualizations.

---


## 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor key business KPIs.
- Compare sales across different cities.
- Identify top-performing products and customers.
- Analyze category-wise sales.
- Evaluate delivery performance.
- Build an interactive dashboard for business analysis.
- Support data-driven decision-making.

## 📊 Dashboard Features

The dashboard includes the following visualizations:

- 📍 Sales Across Regions/Cities
- 🛍️ Total Sales by Categories
- 👤 Top 10 Customers
- 📦 Top 10 Products
- 📉 Bottom 10 SKUs
- 🥧 Customer Sales Contribution
- 📅 Monthly Sales Trend

---

## 🎛 Interactive Filters (Slicers)

The dashboard includes interactive slicers for:

- Category
- Payment Method
- City
- Customer Segment
- Customer Type

These slicers dynamically filter all KPIs and charts.
